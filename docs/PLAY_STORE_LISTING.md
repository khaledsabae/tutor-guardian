# المربّي الذكي — نصوص Play Store (ASO)

نصوص محسّنة للوصف، غنية بالكلمات المفتاحية التربوية، توضّح أنه **منهج متكامل** (لا تطبيق نصائح سريع) و**مجاني لوجه الله**. الصق في Play Console → Store listing.

---

## الوصف القصير (Short description — ≤ 80 حرفًا)
```
منهج تربية إسلامية متكامل لطفلك: مسارات، بودكاست، فيديو، ومساعد ذكي — مجاني لوجه الله
```

## الوصف الكامل (Full description)
```
«المربّي الذكي» ليس مجرد تطبيق نصائح — بل رحلة تربية متكاملة ترافق طفلك خطوة بخطوة، من الحمل وحتى ١٨ سنة. منهجٌ تربوي إسلامي عملي تعيشه مع طفلك، لا تقرؤه في دقائق.

🤍 مجاني بالكامل — صدقة لوجه الله، بلا إعلانات ولا اشتراكات.

📚 منهج متكامل لكل مرحلة عمرية
مسارات تربوية من ٢٨ يومًا لكل فئة عمرية (الحمل–سنة، ٢-٣، ٤-٦، ٧-٩، ١٠-١٢، ١٣-١٥، ١٦-١٨)، تغطي:
• التربية الإسلامية وغرس الإيمان والأخلاق
• تطوّر الطفل وسلوكه ونموّه
• الأمان الرقمي وإدارة وقت الشاشة
• صحة الطفل ونموّه الجسدي

🎧 محتوى تعيشه بطرق متعددة
دروس مكتوبة + بودكاست صوتي + فيديو تعريفي لكل مسار + بطاقات تعليمية + اختبارات تفاعلية.

🌟 رحلة طفلك
سجّل محطات نموّ طفلك الإيمانية (أول صلاة، أول صيام، حفظ القرآن…)، وتابع تقدّمه عبر الزمن في سجلٍّ تعتزّ به.

💬 مساعد تربوي ذكي
إجابات موثوقة عن تحدياتك التربوية اليومية وقت ما تحتاج.

📖 ركن القرآن والأذكار
قراءة وتلاوة القرآن، وأذكار وأدعية للطفل.

الكلمات المفتاحية: تربية الأطفال، تربية إسلامية، نصائح تربوية، تربية الطفل، تطور الطفل، سلوك الطفل، تأديب الأطفال، الأبوة والأمومة، صحة الطفل، أمان رقمي، وقت الشاشة، القرآن للأطفال، أذكار، أدعية، أخلاق الطفل، مربي، الحمل، الرضاعة، المراهقة.

ابدأ رحلتك مع «المربّي» اليوم — واجعل تربية طفلك عبادةً ومتعة.
```

## ملاحظات ASO إضافية (من تقرير المختبرين)
- **سكرينشوتات**: استبدل الـscreenshots العامة بصور تُبرز المميزات (المسارات، رحلة الطفل، البودكاست/الفيديو، المساعد) مع نصوص توضيحية فوقها.
- **عنوان التطبيق**: ممكن «المربّي — تربية إسلامية متكاملة» لإضافة كلمات مفتاحية في العنوان.
